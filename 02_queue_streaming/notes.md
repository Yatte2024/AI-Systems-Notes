# AI Systems Engineering Notes

A personal learning series about modern AI systems, agent workflows, and AI engineering infrastructure.

The focus is not only on models themselves, but also on the surrounding systems that make modern AI applications work in practice.

Topics include:

- async workflows
- queues
- streaming
- agent orchestration
- memory systems
- retrieval pipelines
- evaluation workflows
- observability
- tool execution
- multi-agent coordination
